    Suites et python

    Suite définie de façon explicite

    Pour déterminer, dans le cas d'une suite définie explicitement, il suffit d'utiliser la même méthode que pour calculer une image d'une fonction.

    def f(x):
    return -5+7*x

def suiteU(n):
    return f(n)

n=5
print("u("+str(n)+") = "+str(suiteU(5)))

    affiche

    u(5) = 30

    L'utilisation d'une fonction python pour f simplifie le calcul pour une autre suite : il suffit de modifier cette fonction python.

    Somme des termes d'une suite

    Pour calculer une somme de termes, il faut une variable qui va accumuler en les ajoutant les termes au fur et à mesure.

    def f(x):
    return -5+7*x

def suiteU(n):
    return f(n)

def somme(n):
    S=0
    for i in range(n+1):
        S=S+suiteU(i)
    return S

n=10
print('Sa somme u(0)+...+u('+str(n)+') = ',somme(n))
    Exercice

    Soit

    \left\{ \begin{array}{lcl} u_0 = 3000 \\ u_{n+1} = 0.9u_n+100 \end{array} \right.

    Ecrivez le programme qui retourne u_0+u_1+\ldots+u_{n} en fonction de n.
